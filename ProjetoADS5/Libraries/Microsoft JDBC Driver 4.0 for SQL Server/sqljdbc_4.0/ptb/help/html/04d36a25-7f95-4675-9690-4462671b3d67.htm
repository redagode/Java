Método prepareCall (java.lang.String, int, int)

Cria um objeto SQLServerCallableStatement que gera os objetos SQLServerResultSet com tipo e simultaneidade fornecidos.

Sintaxe

public java.sql.CallableStatement prepareCall(java.lang.String sql,
                                              int resultSetType,
                                              int resultSetConcurrency)

Parâmetros

sql

Uma String que contém uma instrução SQL.

resultSetType

Um int que indica o tipo de conjunto de resultados.

resultSetConcurrency

Um int que indica o tipo de simultaneidade do conjunto de resultados.

Valor de retorno

Um objeto CallableStatement.

Exceções

SQLServerException

Comentários

Esse método prepareCall é especificado pelo método prepareCall na interface java.sql.Connection.
